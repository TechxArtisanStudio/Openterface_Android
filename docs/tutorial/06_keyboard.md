# Tutorial 6 — Using the On-Screen Keyboard

**Level:** Beginner · **Prerequisites:** Tutorial 4 · **Estimated time:** 15 minutes

The on-screen keyboard lets you type on the target computer without a physical keyboard. It supports regular keys, function keys, shortcuts, and modifier combinations.

---

## 6.1 Opening the Keyboard

Tap the **Keyboard button** (⌨) at the bottom-right of the main screen. The keyboard panel appears above the buttons.

To close it, tap the **X** button on the keyboard, or tap the keyboard button again.

---

## 6.2 The Keyboard Toolbar

The top row of the keyboard has these controls:

| Button | What It Does |
|---|---|
| **⋮⋮ (drag handle)** | Drag to move the keyboard panel up or down |
| **Zoom In / Zoom Out** | Make the keyboard keys bigger or smaller |
| **ShortCut** | Switch to the shortcut panel (Ctrl+C, Win+L, etc.) |
| **Function** | Switch to the function keys panel (F1–F12, navigation) |
| **Ctrl** | Toggle Ctrl modifier for the next key |
| **Shift** | Toggle Shift modifier for the next key |
| **Alt** | Toggle Alt modifier for the next key |
| **Win** | Toggle Windows/Command key for the next key |
| **System** | Switch to system keys (symbols, arrows, Enter, Backspace) |
| **X (close)** | Close the keyboard |

---

## 6.3 Typing Letters

When the system key panel is active, you'll see a QWERTY layout with:

- **Letters** (A–Z)
- **Numbers** (0–9) with their shift symbols (!, @, #, etc.)
- **Punctuation** (-, +, [, ], :, ", and more)
- **Backspace** and **Enter** keys

Tap any key to send that character to the target computer.

---

## 6.4 Modifier Keys — How They Work

The modifier keys (Ctrl, Shift, Alt, Win) are **toggle buttons**:

1. Tap **Ctrl** → it highlights (now active)
2. Tap a letter, say **C** → the app sends **Ctrl+C** to the target
3. The modifier **resets automatically** after the key is sent

To send a multi-modifier combination:

1. Tap **Ctrl** (highlights)
2. Tap **Alt** (both highlight)
3. Tap **Del** → sends **Ctrl+Alt+Del**
4. All modifiers reset

---

## 6.5 Shortcut Panel

Tap **ShortCut** to see pre-built keyboard shortcuts:

| Shortcut | What It Does |
|---|---|
| **Ctrl+C** | Copy |
| **Ctrl+V** | Paste |
| **Ctrl+Z** | Undo |
| **Ctrl+X** | Cut |
| **Ctrl+A** | Select All |
| **Ctrl+S** | Save |
| **Win+Tab** | Task View (Windows) |
| **Win+S** | Search (Windows) |
| **Win+E** | File Explorer (Windows) |
| **Win+R** | Run dialog (Windows) |
| **Win+D** | Show Desktop (Windows) |
| **Win+L** | Lock Screen (Windows) |
| **Alt+F4** | Close Window |
| **Ctrl+Alt+Del** | Security Screen (Windows) |
| **Alt+PrtScr** | Screenshot Active Window |

These are single-tap — no need to toggle modifiers first.

---

## 6.6 Function Keys Panel

Tap **Function** to access:

- **F1–F12** — standard function keys (useful for BIOS setup, help menus)
- **PrtSc, ScrLk, Pause** — system keys
- **Ins, Home, PgUp, Del, End, PgDn** — navigation
- **Esc, Tab, Tilde (~)** — special keys
- **Arrow keys** — up, down, left, right

---

## 6.7 Keyboard Layouts

The app supports different regional keyboard layouts. If you're using a non-US layout (e.g., Japanese JIS, German QWERTZ), make sure the correct layout is selected in the settings so that keys map correctly on the target.
