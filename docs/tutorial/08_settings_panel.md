# Tutorial 8 — The Settings Panel

**Level:** Beginner · **Prerequisites:** Tutorial 4 · **Estimated time:** 10 minutes

The settings panel is your control center. Every configuration option lives here.

---

## 8.1 Opening the Panel

Tap the **Menu button** (☰) at the bottom-right of the main screen. The panel slides in from the right side.

To close it, tap anywhere in the empty area outside the panel, or tap the menu button again.

---

## 8.2 Every Setting Explained

| Setting | What It Does |
|---|---|
| **Device** | Shows connected USB devices and their status. Tap to select a device or re-scan. |
| **Disconnect Device** | Safely releases all USB connections. Use this before unplugging. |
| **Baudrate** | Sets the serial communication speed. Use **115200** (default) unless you have issues, then try **9600**. |
| **Controls** | Camera brightness, contrast, and hue sliders. |
| **Video Format** | Resolution and frame rate selector. |
| **Rotate 90 CW** | Rotate the video clockwise. |
| **Rotate 90 CCW** | Rotate the video counter-clockwise. |
| **Flip Horizontally** | Mirror the video left-to-right. |
| **Flip Vertically** | Mirror the video top-to-bottom. |
| **Screen Capture** | Take a screenshot of the current frame. |
| **Record Video** | Start/stop video recording. |
| **Relative / Absolute / Drag** | Switch mouse control mode. |
| **About Device** | Shows Android version and app version info. |

---

## 8.3 Device Management

Tap **Device** to see:

- **Connected devices** — a list of USB devices the app has detected
- **Device status** — whether each device is connected, active, or disconnected
- **Re-scan** — refresh the list if you plugged in a device after opening the app

If you have multiple Openterface devices connected (rare), you can select which one to use here.

---

## 8.4 Baudrate

The baudrate controls how fast the app communicates with the Openterface device over the serial connection:

- **115200** — the default and recommended speed. Works for almost everyone.
- **9600** — a slower speed. Only use this if you experience communication problems at 115200.

To change: tap **Baudrate** → select the speed → tap **OK**.

---

## 8.5 About Device

Shows:

- Your Android version
- The app's version number

This is useful when reporting bugs or checking if you have the latest version.
