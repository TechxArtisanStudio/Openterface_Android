# Tutorial 12 — Everyday Tasks and Tips

**Level:** All levels · **Prerequisites:** Tutorials 1–11 · **Estimated time:** 10 minutes

Practical tips for getting the most out of the app in your daily workflow.

---

## 12.1 For Beginners

- **Start with Absolute mouse mode** — it's the most intuitive. Tap and the cursor goes there.
- **Use the shortcut panel** — for common actions like Ctrl+C/V or Win+L, it's faster than pressing modifier keys individually.
- **Adjust brightness and contrast** — if the video looks too dark or washed out, the target's display settings may differ from what your phone expects. A quick adjustment usually fixes it.
- **Keep the app updated** — patch releases fix bugs you might not even know you're hitting.

---

## 12.2 For Regular Use

- **Learn the shortcuts** — Win+L to lock the target, Ctrl+Alt+Del for Windows login, Alt+F4 to close a frozen app on the target. These are all one tap away in the ShortCut panel.
- **Use function keys for BIOS** — F2, F10, F12, or Del during boot to enter BIOS/UEFI settings on the target machine.
- **Record when troubleshooting** — if you're diagnosing a problem on the target, start a video recording. You can review it later to catch things you missed in real-time.
- **Rotate the video** if your target is in portrait orientation.

---

## 12.3 For Power Users

- **Absolute Drag mode** is great for precise tasks — selecting text, dragging files, resizing windows on the target.
- **Pinch-to-zoom** the preview to see small details more clearly, then tap precisely.
- **Lower the resolution temporarily** if you're experiencing lag — you can always increase it again once you're done with the heavy lifting.
- **Use the Disconnect button** before unplugging — it ensures the target computer properly releases the keyboard and mouse.

---

## 12.4 Common Workflows

### Server Management

1. Connect to the target server via Openterface
2. Use the on-screen keyboard to send **Ctrl+Alt+Del** to reach the login screen
3. Type credentials using the keyboard
4. Use mouse to navigate the desktop
5. Take screenshots of important configurations

### BIOS / Boot Configuration

1. Connect and power on the target
2. Use the **Function** key panel to press **F2** or **Del** during boot
3. Navigate BIOS with the mouse and keyboard
4. Change boot order, enable settings, save and exit

### Remote Troubleshooting

1. Connect to a misbehaving target
2. Start **video recording**
3. Use keyboard shortcuts to open terminal/command prompt
4. Run diagnostics while recording
5. Stop recording and review the footage
